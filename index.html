Arrays

Los arrays son colecciones de elementos de cualquier tipo, ya sean datos numericos,booleanos o tambien porque no objetos y funciones

Para crear un Array se utiliza la notacion let:

let number = [ 2 , 3 , 4 , 5 ];

Ahora si queremos ver la longitud de nuestro array en la consola

console.log( number.length );

Los elementos estan ubicados en una posicion numerada dentro del array comenzando desde cero. Para acceder a una posicion y mostrarla en consola

console.log(number[ posicion ]);

Para entrar al ultimo usamos length - 1

console.log( number.length-1 );

Otra manera de acceder a los elementos de un array es con el metodo at que nos permite acceder a los elementos indicandole indices de la misma manera que la notacion comun, la unica particularidad es que podemos darle indices negativos y recorre el array alrevez

console.log(number.at(0));

Para recorrer un arreglo podemos usar un un for

for (i=0 ; i<num.length ; i++) {
   console.log(number[ i ])
}

Vamos a insertar elemetos a un array con el metodo push, este metodo los agrega al final del array

number.push( 6 )

Y este al principio

number.unshift("negra modelo")

Si quisieramos agregar o eliminar algun dato de alguna posicion especifica del array usamos el metodo splice.
Este metodo utiliza tres parametros, el primero para indicar la posicion del elemento a agregar, el segundo cuantos eliminar a partir de esa posicion y el tercero que es lo que se va a agregar

number.splice( 2 , 0 ,"er" );

Ahora eliminaremos algo del array con el metodo pop, este metodo elimina el ultimo valor y lo devuelve, si quiero lo guardo asignando el metodo a una variable y lo muestro por consola

let a = number.pop()
console.log(number)
console.log(a)

Pasa lo mismo con el metodo shift, la diferencia es que lo hace desde el principio del array

let a = number.shift()
console.log(number)
console.log(a)

Ahora concatenaremos array con el metodo concat, para ello crearemos un nuevo array y guardaremos la concatenacion alli

let combinado = number.concat(strings);
console.log(combinado)

Para buscar un elemento determinado se usa indexOf, esto nos devuelve un buleano para indicarnos que existe o no ese elemento buscado dentro del array.
Hay que tener en cuenta que si se busca un string hay que usar comillas.

console.log(combinado.indexOf("snacks"))

El metodo slice sirve para armar otro array a partir de uno que ya tengamos creado. Igualamos el metodo a una variable y nos guarda la nueva cadena, este metodo acepta dos parametros, desde donde empieza a tomar los elemetos y el segundo parametro donde los corta

let combinumber = combinado.slice(3,6)
console.log(combinumber)

El metodo fill me cambia los valores del array, acepta tres parametros el primero es con lo que se va a llenar, segundo es donde empieza a llenar y el tercero donde termina

combinado.fill( gato,1,4 )

Ahora si queremos copiar los valores de un array podemos usar el metodo spread

numberCopy = [...number]
console.log(numberCopy)

De la misma marea spread sirve para concatenar dos array de strings

let numerosCombinados = [...number,...combinado]
console.log(numerosCombinados)

number = [1,2,3,4]

return a+b+c+d
